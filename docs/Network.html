Throws:
  • Error when trying to add existing edge
  • Error when trying to add edge over allowed limit
  • Error when trying to add edge on inexistent node (if not forcing edge)
Example

Creating a Network net and adding 2 nodes and an edge between them.

import { Network } from './network.js'
const net = new Network();
// Adding nodes
net.addNode({ id: 'A' });
net.addNode({ id: 'B' });
// Adding edge
net.addEdge({ id: 'coolEdge', nodeA: 'A', nodeB: 'B' })
